# Ciência de Dados — 2026.1

Este repositório reúne os materiais, códigos e atividades desenvolvidos na disciplina **Ciência de Dados**, ofertada no semestre **2026.1**.  
O foco da disciplina é apresentar conceitos fundamentais e práticos de Ciência de Dados, desde a análise exploratória até técnicas de aprendizado de máquina.

---

## Unidade I — Fundamentos e Análise de Dados
- Revisão de Python  
- Pandas para manipulação de dados  
- Revisão de Estatística Descritiva  
- Análise Exploratória de Dados (EDA)  
- Visualização de dados  

---

## Unidade II — Inteligência Artificial e Aprendizado de Máquina
- Introdução à Inteligência Artificial  
- Introdução ao Aprendizado de Máquina  
- Pré-processamento de Dados:
  - Tratamento de dados ausentes  
  - Tratamento do desbalanceamento de classes  
  - Transformação de variáveis categóricas e numéricas  
  - Normalização de dados  
- Aprendizado Supervisionado:
  - Classificação  
  - Regressão  

---

## Unidade III — Técnicas Avançadas e Extração de Dados
- Aprendizado Não Supervisionado:
  - K-Means  
  - DBSCAN  
- Regras de Associação  
- Data Scraping:
  - Extração de dados da web com Python  

---

## Objetivo da Disciplina
Capacitar o aluno a compreender, analisar e extrair conhecimento de dados, utilizando técnicas estatísticas, visualização e métodos de aprendizado de máquina aplicados a problemas reais.

---

## Tecnologias Utilizadas
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Scikit-learn  
- Jupyter Notebook  

---
